## 🧰 Dev Toolbox 的主要功能

Dev Toolbox 提供了多种实用工具，涵盖 **前端、后端、DevOps 和数据处理** 等领域，包括：

### 🔹 **1. 代码格式化与转换**
- 自动格式化代码（JSON、SQL、XML、HTML、CSS、JavaScript 等）
- 支持多种格式转换（CSV ↔ JSON、YAML ↔ JSON 等）

### 🔹 **2. 文本与字符串工具**
- Base64 编码/解码、URL 编码/解码
- 哈希生成器（MD5、SHA1、SHA256 等）
- 正则表达式测试工具（Regex Tester）

### 🔹 **3. 图片与文件工具**
- 图片压缩、调整大小、格式转换
- 文本或数据文件格式互转

### 🔹 **4. 时间与日期工具**
- 时间戳 ↔ 日期时间 转换  
- 显示 UTC、ISO8601 等格式，用于 API 调试

### 🔹 **5. 其他实用工具**
- UUID 生成器  
- 随机字符串/密码生成器  
- 进制转换器（二进制、十六进制、十进制）

所有工具都经过优化，**运行快速、界面简洁**，在 **桌面端和移动端** 都能流畅使用。
